pidgin_plugin_lol_chat
======================

Pidgin plugin for League of Legends Chat. This plugin help you read user-information more comfortable.
Works only with english localization.

HOW TO INSTALL

DEBIAN JESSIE

chmod +x make.sh

BUILD
./make.sh
BUILD DEBUG
./make.sh debug
INSTALL
./make.sh install
REMOVE BUILD
./make.sh clean
UNINSTALL
./make.sh uninstall
